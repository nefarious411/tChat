Welcome to tChat.

This project is intended as a Twitch.tv chat replacement. The development is still in early stages, but the goals are as follows:

  
    
    
  • Command line bot support
    • 
    
  • GUI for chat replacement
    • 
    
  • Support for multiple logins.  Ability to login for yourself and additional login for bots
    • 
    
  • realtime bot control
    • 
    
  • plugin support.  Give ability to add addition plugins with easy to use API
